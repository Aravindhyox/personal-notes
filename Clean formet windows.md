# Format Drive / USB / Windows clean install:

<h3>cmds:</h3>

<li>Diskpart</li>
<li>List disk</li>
<li>select disk 0,1,2</li>
<li>clean</li>
<li>create partition primary</li>
<li>select partition primary</li>
<li>format fs=ntfs quick</li>
<li>assin</li>
<li>exit</li>
<li>exit</li>

<h3>This cmds works only on windows </h3>
